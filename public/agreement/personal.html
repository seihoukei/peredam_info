Использование персональных данных сервисом Передам.инфо

Данный сервис запрашивает и использует персональные данные пользователей исключительно для выполения своих функций (передача показаний счетчиков и контроль сроков передачи), только с их согласия и только в необходимых объемах, а именно:

  • Номер телефона (не обязателен) - для идентификации пользователя, в будущем - возможности восстановления доступа. В дальнейшем также возможно появится возможность подписаться на рассылку СМС-напоминаний.
  • Адрес электронной почты (не обязателен) - для идентификации пользователя, возможности восстановления доступа и подписки на напоминания о сроке передачи показаний.
  • Имя пользователя (не обязательно) - для идентификации пользователя
  • Номера лицевых счетов и используемых счетчиков, а так же их показания и время их передачи - для передачи показаний поставщикам услуг в требующемся объеме, слежения за соблюдением сроков передачи показаний ив будущем - для статистики
  • Автоматически формируемые браузером данные для получения Push-уведомлений - для напоминаний о сроках передачи показаний.

Данные пользователей хранятся в базе данных на серверах в дата-центре хостинга "1Gb.ru" в Москве и передаются по сети через защищенное HTTPS-соединение.

Пользователь может отозвать разрешение на использование своих персональных данных в любой момент, отправив электронное письмо на адрес support@peredam.info с указанием имени пользователя или телефона, указанного при регистрации, и подтверждением его принадлежности. При этом данные о пользователе, его системах, настройках, контактах и переданных показаниях будут удалены без возможности восстановления.
